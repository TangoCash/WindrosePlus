Character Repair

Back to dashboard

This repairs the known Windrose progression drift where a character crashes a server on join with RewardLevel < CurrentLevel. It works on a copy of your uploaded save and returns a repaired zip.

Before You Upload

  1. Close Windrose completely. Check Task Manager if needed.
  2. Open %LOCALAPPDATA%\R5\Saved\SaveProfiles\.
  3. Right-click your long-number SteamID folder and create a zip.
  4. Upload that zip here. Do not upload the server world save.
  5. After download, extract the repaired zip back into SaveProfiles and replace existing files.

If Steam Cloud asks which files to keep, keep the local files you just replaced. The tool refuses unsafe cases instead of guessing.

Repair Upload

Download repaired save again

Maximum upload size is 200 MB. Safe mode only fixes no-spend progression drift. If the save has spent-point drift or a different corruption type, this page will stop and tell you no safe automatic repair was made.
